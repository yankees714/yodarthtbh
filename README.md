yodarthtbh
==========

### Setup
Install reqs with:

```
pip install -r reqs.txt
```

Create the file `settings.py` to hold your keys, like:

```python
TWITTER_CONSUMER_KEY = ""
TWITTER_CONSUMER_SECRET = ""
TWITTER_ACCESS_TOKEN = ""
TWITTER_ACCESS_TOKEN_SECRET = ""

YO_TOKEN = ""
```

### Usage
```
python app.py
```
